2's Complement. What is it?

I. How is number represented in a computer?

 Due to a computer's limitation, a computer cannot count in base 10 as we only have 2 states to work with (ON and OFF). As such, computer counts in base 2 (each digit represent the amount of 2 to the power of digit place instead of 10). As such, 10 in base 2 would become 1010, meaning 0 * 20 + 1 * 21 + 0 * 22 + 1 * 23 = 2 + 8 = 10.

II. How does computer represent negative number?

 In order to represent negative number using binary and a limited number of bits, we use a system called 2's complement.

 In short: The left most bit of a binary number represent its sign:

  - 0 for positive

  - 1 for negative

III. How do you get the 2's complement of a specific number?

 To get the 2's complement of a number, you flip all of the number's bits from 1 --> 0 and 0 --> 1 and add 1 to the new number.

 Example: to get the 2's complements of the number 11.

  - Binary representation of the number 11 in a 32 bits system is: 0000 0000 0000 0000 0000 0000 0000 1011

  - Step 1: Flip all of the number's bit 1 --> 0 and 0 --> 1: 1111 1111 1111 1111 1111 1111 1111 0100

  - Step 2: Add 1 to the new number: 1111 1111 1111 1111 1111 1111 1111 0101

  - Thus 1111 1111 1111 1111 1111 1111 1111 0101 represents -11.

  - To verify you add the original number and the new number together: 0000 0000 0000 0000 0000 0000 0000 1011 + 1111 1111 1111 1111 1111 1111 1111 0101 = 0000 0000 0000 0000 0000 0000 0000 0000 (11 + (-11) = 0)
